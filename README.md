## Overview
This project analyzes Growth vs Value mutual fund performance
using statistical and financial metrics.

- Skewness & Kurtosis
- Return Distribution Visualization
- Cumulative Returns

## Assumptions
Risk-free rate assumed: 2% annually.

## Tools
Python, NumPy, SciPy, Matplotlib

## How to Run
pip install -r requirements.txt
